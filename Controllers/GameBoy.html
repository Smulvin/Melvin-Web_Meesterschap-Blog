Opdracht

Tijdens dit project moest er vooral gelet worden op toegankelijkheid, de huisstijl en dat het correct en netjes ingevuld kon worden.

Deze punten ging mij best goed af, ik had alleen een beetje moeite met responsiveness om dat te doen zonder media queries. Maar het is aardig goed gelukt bij deze opdracht, want je kan het flink goed onder elkaar zetten op mobiel formaat.

Weekly Nerd

Halverwege het project kregen we ook nog een weekly nerd van Robert Broersma en Yolijn van der Kolk. Hier heb ik nog punten uit kunnen halen om mijn formulier te verbeteren, zoals het hebben van een introductie pagina.

Toegankelijkheid

Dit was mijn eerste introductie tot toegankelijkheid, zo heb ik extra focus gelegd op dat mijn html goed opgebouwd is, maar ook dat het volledige formulier volledig ingevuld kan worden zonder muis.
