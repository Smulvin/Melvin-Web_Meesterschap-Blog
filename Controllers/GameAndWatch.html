Hackathon

Dit was een kort groepsproject dat we met 4 mensen samen deden. Tijdens het project heb ik gewerkt aan assets, easter eggs, effecten en een minigame.

Assets

Aangezien we tijdens de minor bijna alleen maar code schrijven, vond ik het wel fijn om ook wel eens even wat assets in Figma te maken voor de verandering. Ik heb assets gemaakt van de aarde, ufo, raket, satteliet, maan, mars sterren en alles voor de minigame.

Easter Eggs - UFO

Ik heb een animatie gemaakt voor een ufo die af en toe langskomt, achter de ufo hangt een banner met het logo van het bedrijf. Dit was een creatieve manier om het logo te verwerken in de website.

Easter Egg - Cat

Om gewoon een kleine easter egg toe te voegen die niks extra biedt, maar wel de website meer herinnerbaar maakt, had ik een nyan cat animatie toegevoegd met sfx. Deze gebruikte bijna dezelfde aniamtie als de ufo.

Effect - Black Hole

Om een easter egg toe te voegen maar ook de website leuker te maken en meer interactie te geven hadden we wat effecten toegevoegd. Zelf had ik een black hole effect gemaakt gebasseerd op een fish lens.

Effect - Alien

Een ander effect dat ik had toegevoegd aan dit project was het alien effect. Bij dit effect komt er een particle system aan aliens uit de je muis, deze is dus ook goed te combineren met andere effecten.

Game - Space Invaders

Om een beetje eigen identiteit aan het project toe te voegen had ik een kleine game toegevoegd. Deze was geinspireerd op space invaders. Wat tevens ook onze teamnaam was. Dit vond ik het leukste om te maken.

Reflectie

Tijdens dit project heb ik goed kunnen samenwerken met mijn team. Ik merkte dat ik vaak met ideeën kwam voor de easter eggs, wat ik zelf ook leuk vond om te maken. Daarbij heb ik veel kunnen oefenen met animaties tijdens dit project. Ik had soms wel nog een beetje moeite met mergen van code, maar qua samenwerking was ik blij verrast hoe soepel alles ging.
