Opdracht

Voor deze opdracht was het de bedoeling om een Silly Walk te maken, of in mijn geval een Silly Swim. De uitdaging zit hem in dat je geen JavaScript mag gebruiken.

OctoDad

Om er een beetje een game thema aan te geven had ik gekozen om OctoDad na te maken, een karakter dat van zichzelf al raar beweegt. Ik vond dit een van de leukste vakken van de minor. Ik vond het een leuke uitdaging dat je één van de talen niet mocht gebruiken, dit zorgde ervoor dat je anders ging denken en op nieuwe ideeën kwam voor het probleem.

Ideeën

Gedurende het project kreeg ik steeds meer leuke ideeën om toe te voegen. Het leukste idee vond ik hoe de armen bewogen. Ik had radio buttons gebruikt om te zien waar een gebruiker heeft geklikt om op die manier de armen de juiste kant op te laten bewegen. Met JavaScript had dit veel makkelijker gekund, maar deze oplossing vond ik zelf leuker.

Reflectie

Tijdens dit project heb ik goed kunnen samenwerken met mijn team. Ik merkte dat ik vaak met ideeën kwam voor de easter eggs, wat ik zelf ook leuk vond om te maken. Daarbij heb ik veel kunnen oefenen met animaties tijdens dit project. Ik had soms wel nog een beetje moeite met mergen van code, maar qua samenwerking was ik blij verrast hoe soepel alles ging.
