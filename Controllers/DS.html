Voor het vak CSS mochten we geen gebruik maken van JavaScript, maar toch zo veel mogelijk interactie proberen te maken op een website. Ik had gekozen om de Silly Walk opdracht te doen, maar in plaats van lopen, zou mijn karakter zwemmen. Om er een beetje een game thema aan geven koos ik voor Octodad, een karakter dat van zichzelf al raar beweegt. Ik vond dit een van de leukste vakken van de minor. Ik vond het een leuke uitdaging dat je één van de talen niet mocht gebruiken, dit zorgde ervoor dat je anders ging denken en op nieuwe ideeën kwam om een probleem op te lossen. Het leukste idee waar ik op kwam is om radio buttons te gebruiken om te zien waar een gebruiker heeft geklikt om op die manier de armen de juiste kant op te laten bewegen. Met JavaScript had dit veel makkelijker gekund, maar deze oplossing vond ik zelf leuker.
Tijdens dit vak heb ik vooral veel kunnen werken voor mijn eerste leerdoel, het veel gebruik maken van animaties. Zo had ik een animatie voor de benen, die ik per been anders maakte door gebruik van animation-delay. Maar ook zat er een animatie op de armen en bubbels. Allemaal op de website levendiger te maken.
